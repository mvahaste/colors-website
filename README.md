# SE-22 veebilehe projekt

### Veebilehest

Veebilehe teemaks on värvid. Värve on võimalik otsida nii hex koodi kui ka nime järgi ja saab ka mingil määral _pixel art_-i teha.

Ma teadsin, et tahan kasutada _grid_-i, ja idee tuli sellest, et alguses panin ajutiselt _grid_-i elementideks värvid, kuid ei osanud neid millegagi asendada ja mulle nii ka sobis, seega valisingi teemaks värvid.

Leht on inglise keeles, sest olen nii harjunud ja minu arvates näeb see ka parem välja.
